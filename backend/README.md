BugHunter – AI-Powered Code Review & Repository Analysis Platform

Built a full-stack application using Next.js, FastAPI, Gemini AI, and GitHub APIs.
Developed AI-based code review and bug detection features with automated fix suggestions.
Implemented GitHub repository analysis by fetching source files and generating architecture, security, performance, and code-quality reports.
Generated downloadable PDF reports for repository assessments.
Integrated Gemini AI for intelligent repository scoring and recommendations.
